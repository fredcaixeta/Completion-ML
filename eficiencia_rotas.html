Eficiência na Rota de Entregas - Otimização de Operações Logísticas com Machine Learning¶

Notebook criado para o Desafio Técnico.

Das variáveis entregues no problema, decidi usar apenas as descritas abaixo. Pois são as relevantes para o a estimativa - Tempo Estimado de Viagem - foco do Modelo de Machine Learning - prever qual o tempo necessário para levar o item do ponto de partida ao destino de entrega, com as dependências do ambiente.

  • Destino da Entrega: Diferentes locais de entrega (Local A, Local B, etc.)
  • Distância: Distância entre o ponto de partida e o destino
  • Tempo Estimado de Viagem: Tempo estimado de viagem
  • Condições de Tráfego: Categorias de tráfego (Baixo, Médio, Alto)

Workflow:¶

  1. Codificar os valores possíveis para 'Local de Partida', 'Destino da Entrega' e 'Condições de Tráfego';

  2. Configurar para 'target' de predicção do modelo - 'Tempo Estimado de Viagem'

  3. Configurar o Modelo - Random Forest Regressor - e analisar precisão

    'Condições de Tráfego': np.random.choice(['Baixo', 'Médio', 'Alto'], 100)
})

Escolha do Modelo

Por haver algumas features com informações não-lineares, decidi utilizar o Random Forest Regressor, que utiliza árvore de decisão, sem envolver algoritmos lineares.

Início do Preparo do Modelo:

Usando Label Encoder e StandardScaler para separar cada categoria em códigos numéricos e escalar os valores, respectivamente. Escalar valores aqui é importante, pois as features são muito distintas entre si (Destino de Entrega e Distância, por exemplo).

from sklearn.preprocessing import LabelEncoder, StandardScaler
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestRegressor
from sklearn.metrics import mean_squared_error

# Codificar variáveis categóricas
label_encoders = {}
for column in ['Local de Partida', 'Destino da Entrega', 'Condições de Tráfego']:
    le = LabelEncoder()
    entregas_data[column] = le.fit_transform(entregas_data[column])
    label_encoders[column] = le

# Normalizar Distância
scaler = StandardScaler()
entregas_data[['Distância']] = scaler.fit_transform(entregas_data[['Distância']])

# Definir o que é feature e o que é target
X = entregas_data[['Local de Partida', 'Destino da Entrega', 'Distância', 'Condições de Tráfego']]
y = entregas_data['Tempo Estimado de Viagem']

# Dividir os dados em conjuntos de treino e teste
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=2)

# Treinar o modelo
model = RandomForestRegressor(n_estimators=100, random_state=2)
_ = model.fit(X_train, y_train)

# Avaliação do Modelo
y_pred = model.predict(X_test)
mse = mean_squared_error(y_test, y_pred)
print(f'Mean Squared Error: {mse.round(2)}')

Mean Squared Error: 5543.8
