Das variveis entregues no problema, decidi usar apenas as descritas abaixo. Pois so as relevantes para o a estimativa - Tempo Estimado de Viagem - foco do Modelo de Machine Learning - prever qual o tempo necessrio para levar o item do ponto de partida ao destino de entrega, com as dependncias do ambiente.

  • Destino da Entrega: Diferentes locais de entrega (Local A, Local B, etc.)
  • Distncia: Distncia entre o ponto de partida e o destino
  • Tempo Estimado de Viagem: Tempo estimado de viagem
  • Condies de Trfego: Categorias de trfego (Baixo, Mdio, Alto)

Workflow:

  1. Codificar os valores possveis para 'Local de Partida', 'Destino da Entrega' e 'Condies de Trfego';

  2. Configurar para 'target' de predico do modelo - 'Tempo Estimado de Viagem'

  3. Configurar o Modelo - Random Forest Regressor - e analisar preciso

Escolha do Modelo

Por haver algumas features com informaes no-lineares, decidi utilizar o Random Forest Regressor, que utiliza rvore de deciso, sem envolver algoritmos lineares.

Incio do Preparo do Modelo:

Usando Label Encoder e StandardScaler para separar cada categoria em cdigos numricos e escalar os valores, respectivamente. Escalar valores aqui importante, pois as features so muito distintas entre si (Destino de Entrega e Distncia, por exemplo).

from sklearn.preprocessing import LabelEncoder, StandardScaler
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestRegressor
from sklearn.metrics import mean_squared_error

# Codificar variveis categricas
label_encoders = {}
for column in ['Local de Partida', 'Destino da Entrega', 'Condies de Trfego']:
    le = LabelEncoder()
    entregas_data[column] = le.fit_transform(entregas_data[column])
    label_encoders[column] = le

# Normalizar Distncia
scaler = StandardScaler()
entregas_data[['Distncia']] = scaler.fit_transform(entregas_data[['Distncia']])

# Definir o que  feature e o que  target
X = entregas_data[['Local de Partida', 'Destino da Entrega', 'Distncia', 'Condies de Trfego']]
y = entregas_data['Tempo Estimado de Viagem']

# Dividir os dados em conjuntos de treino e teste
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=2)

# Treinar o modelo
model = RandomForestRegressor(n_estimators=100, random_state=2)
_ = model.fit(X_train, y_train)

# Avaliao do Modelo
y_pred = model.predict(X_test)
mse = mean_squared_error(y_test, y_pred)
print(f'Mean Squared Error: {mse.round(2)}')

Mean Squared Error: 5543.8
